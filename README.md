liusidaUI
=============
liusidaUI。适用于手机网站,APP网站等。官方网站：<a href="http://ui.liusida.cn">http://ui.liusida.cn</a>

设计理念
------------------
刘思达UI是一个手机端展示仓库，包含了手机项目中常用的前端样式。目的在于方便以后项目使用。
